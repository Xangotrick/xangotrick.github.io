Functions I

Questions:

Part A

  1. Write a worldprinter() function that prints the string "Hello World!" when called.
  2. Write a printer(astring) function that takes one parameter astring and prints astring when called.
  3. Write a doubleprint(astring) function that takes one parameter astring and prints astring two times in a row.
  4. Write a nprint(astring,n) function that takes two parameters astring and n and prints astring n times.

Part B

  1. Write sum(a,b), a function that returns the sum of a and b.
  2. Write multiply(a,b), a function that returns the a multiplied by b.
  3. What happens if you write print(multiply(sum(2,3),sum(1,2))). Try to understand what is happening by breaking it down step by step.
  4. Write combinator(a,b,c), a function that returns a * c + b * c, by using sum() and multiply().
  5. Write negator(a), a function that returns -1 * a.
  6. What happens when you write print(negator(negator(3))). Again try to understand what is happening here.
  7. Consider the following code block: 
    def multinegator(a,q):
    for i in range(q):
        a = negator(a)
    return a
    Try using this function for a few test cases, then anser the following question.
  8. Without checking with python, try to figure out what multinegator(-1,5) will return. What will multinegator(1,6842688) return?
