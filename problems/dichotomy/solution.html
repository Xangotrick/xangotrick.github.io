Dichotomy

Questions:

  1. Write as variables the parameters of the problem: minint, maxint, inttoguess.
  2. Write a Jane function that take a parameter inputint, and returns information on whether inputint is higher, lower or equal to inttoguess.
    3. def Jane(inputint):
    if inputint == inttoguess:
        return 0
    else:
        if inputint > inttoguess:
            return 1
        else:
            return -1
  3. Write a function that emulates Bob's way of finding the right integer.
    4. def BobGuess():
    max = maxint+1
    min = minint
    guess = int((max - min) / 2 + min)
    found = False
    while not found:
        janeint = Jane(guess)
        if janeint == 0:
            found = True
        else:
            if janeint == 1:
                max = guess
                print(str(guess) + " is too high!")
            else:
                min = guess
                print(str(guess) + " is too low!")
            guess = int((max - min) / 2 + min)
    print(str(guess) + " is good!")
  4. Find the average amount of guesses needed to get the right answer through Bob's method.
