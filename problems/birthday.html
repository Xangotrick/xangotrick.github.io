The birthday paradox

Requirements:

This problem requires knowledge in:
  • - test

Problem:

The odds of that at least two students of a class of 23 is about 50%. The birthday paradox is a counterintuitive, truthful statement. While there is a mathematical way of proving this statement, we would like to find a way of calculating those odds using python.



Write a function that takes a parameter for the amount of students, and outputs the odds of a classroom having at least two students share a birthday.

Questions:

  1. Write a function that returns an array of student birthdays.
  2. Improve on that function by adding a parameter that controls the size of the array.
  3. Write a function that takes an array of children birthdays as a parameter, and returns information on wether or not all birthdays are different
