Circles

Requirements:

This problem requires use of:
  • - Functions
  • - Basic turtle methods
  • - Basic Geometry

Problem:

Using turtle, and drawing only straight lines, we would like to write a function that draws a circle of a given radius.



Questions:

  1. Write a function that will make the turtle draw a square
  2. Improve the previous function, by giving it a parameter sidelength. The sides of the drawn square must now be sidelength pixels long.
  3. Write a function that will make the turtle draw an equilateral triangle.
  4. Improve on it by similarly adding a sidelength parameter.
  5. Write a function with a sidelength parameter that will make the turtle draw a regular pentagon.
  6. Write a function with a sidelength parameter that will make the turtle draw a regular hexagon.

Using turtle, write a method that takes two parameters, nside and sidelength. The method will make the turtle draw a regular polygon with nside sides. Each side will be sidelength pixels long.
