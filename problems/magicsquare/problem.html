Magic Square

Time required: 1H

Requirements:

This problem requires use of:
  • - Functions
  • - Loops
  • - Nested Lists

Problem:

A magic square is a 3x3 grid filled with the numbers 1 through 9. The rules are that all 9 numbers must be used, and the sum of the numbers of any row, column or diagonal must be the same. Through brute force, we will find all possible 3x3 magic squares.



Questions:

  1. Write a variable magic that represents an empty magic square.
  2. Fill magic any way you like. Then write a function ismagic() that returns True if magic is a magic square, or False if not.
  3. Find a way of systematically going through every possible way of filling magic. (Hint: use itertools.permutations())
  4. How many different 3x3 magic squares exist?
